Getting Started

Install the Scratch XML Injector userscript and run your first block injection in under five minutes.

Beginner ~5 min read

01Installation

The injector runs as a userscript — a small piece of JavaScript that your browser executes on specific pages. You need a userscript manager to load it.

Step 1 — Install a userscript manager

A

Tampermonkey (recommended)

Available for Chrome, Edge, Firefox, Safari. Install from your browser's extension store by searching Tampermonkey.

B

Violentmonkey (alternative)

Open-source alternative with the same compatibility. Works identically for this script.

Step 2 — Create the script

1

Open Tampermonkey Dashboard

Click the Tampermonkey icon in your browser toolbar → Dashboard.

2

Create a new script

Click the + tab at the top of the dashboard to open the script editor.

3

Paste the injector code

Delete everything in the editor. Paste the full Scratch XML Injector script. Save with Ctrl+S (or Cmd+S on Mac).

ℹ
The script's @match header already covers scratch.mit.edu/projects/* and turbowarp.org/* — no extra configuration needed.

02Running the Script

1

Open a project in the editor

Go to scratch.mit.edu/projects/<id> and click See inside, or open any project on TurboWarp.

2

Run the script

Click the Tampermonkey icon → find Scratch XML Injector → click it. The panel appears in the top-right corner.

3

Toggle the panel

Running the script again while the panel is open will close it. This is by design — it acts as a toggle.

03The Injector Panel

The panel has three tabs:

TabPurpose
~> InjectPaste XML and inject it into Scratch. Supports both Blocks XML and Project XML modes.
[cp] ExportRead the current sprite's blocks as Blockly XML and copy to clipboard.
? SyntaxQuick in-panel reference for Project XML syntax.

Mode toggle

Within the Inject tab, two mode buttons switch the injector's behaviour:

ModeInputEffect
Blocks XML <xml> … </xml> Replaces the active sprite's block workspace. Non-destructive to other sprites.
Project XML <project> … </project> Wipes the entire project and rebuilds it from scratch. Irreversible.
⚠
Always save first. Project XML mode deletes all existing sprites and cannot be undone. Use File → Save to your computer before building a project.

04Your First Injection

Try this minimal example to confirm everything is working:

1

Open a Scratch project in the editor

Any project works — even a blank one.

2

Select a sprite

Click on Scratch's Sprite 1 (or any sprite) in the sprite list so it's the active editing target.

3

Open the panel and click "! Example"

The example blocks XML is pre-loaded. Make sure the mode is set to ~ Blocks XML.

4

Click "~> Inject"

The blocks appear in the workspace. Press the green flag — the sprite should spin and move.

✓
If blocks appear in the workspace, the injector is working correctly. You're ready to write your own XML.
