Costume Shapes

The injector auto-generates PNG costumes from shape descriptors. No external image files required — just attributes.

Project XML feature

01How It Works

When you use Project XML mode, the injector renders each <costume> element onto an HTML5 Canvas, converts it to a PNG, computes a real MD5 hash of the image data, and registers it with Scratch's asset storage — all in the browser, in real time.

This means you get valid, injected costumes without needing any image file or external upload.

02Circle Costumes

XML
<!-- Solid filled circle -->
<costume name="ball"
         type="circle"
         color="#4c8fff"
         size="60"/>

<!-- Circle with outline stroke -->
<costume name="ball-outlined"
         type="circle"
         color="#ff4488"
         stroke="#ffffff"
         stroke-width="4"
         size="60"/>
name
string
Costume display name shown in the editor.
Default: "costume1"
type
"circle"
Must be exactly circle to render as a circle.
color
CSS hex string
Fill colour of the circle.
Default: #4c4fff
size
number (pixels)
Diameter of the circle in pixels. The canvas is padded to accommodate stroke.
Default: 80
stroke
CSS hex or "none"
Outline colour. Set to none to disable.
Default: none
stroke-width
number (pixels)
Thickness of the outline. 0 means no outline.
Default: 0

03Rectangle Costumes

XML
<!-- Sharp-cornered rectangle -->
<costume name="platform"
         type="rectangle"
         color="#4CAF50"
         width="120"
         height="20"/>

<!-- Rounded rectangle -->
<costume name="card"
         type="rectangle"
         color="#ff4455"
         width="80"
         height="80"
         radius="12"
         stroke="#ff8888"
         stroke-width="3"/>
type
"rectangle"
Must be exactly rectangle to render as a rectangle.
color
CSS hex string
Fill colour of the rectangle.
width
number (pixels)
Width of the rectangle. If 0, falls back to size.
height
number (pixels)
Height of the rectangle. If 0, falls back to size.
radius
number (pixels)
Corner radius. 0 = sharp corners, any positive value = rounded corners.
Default: 0
stroke / stroke-width
string / number
Same as circle — outline colour and thickness.

04Tips & Limitations

ℹ
The rotation center of auto-generated costumes is automatically set to the geometric center of the shape, accounting for stroke width. This matches Scratch's default behavior for most shapes.
⚠
Only circle and rectangle are supported. For arbitrary shapes or real artwork, use the normal Scratch costume editor and only use the injector for block scripts.

If two costumes produce identical PNG data (same attributes), they will share the same asset ID. This is harmless — Scratch handles de-duplication internally.
