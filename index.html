Copy Script

The </> Script tab lets you access and copy the injector's userscript source code without leaving the Scratch editor.

v7.0

01What the Script Tab Does

The </> Script tab is a quick-access panel for the injector's own source code. It's useful when:

A

Sharing the script

You want to copy and send the script to a friend without going to the Tampermonkey dashboard.

B

Installing on another browser

You're on a different computer and want to install the script quickly.

C

Verifying the version

You want to check which version of the script is currently running.

02How to Copy the Script

Method A — From the Script tab (fastest)

1

Open the </> Script tab

Click the </> Script tab in the injector panel.

2

Click "[cp] Copy Script"

The script source or instructions are copied to your clipboard automatically.

3

Or click "Select All" then Ctrl+C

This selects the entire text in the box, which you can then copy manually with Ctrl+C.

ℹ
When the injector runs as a Tampermonkey userscript, browser security prevents the page from reading the script's own injected source code directly. In this case, the Script tab shows instructions pointing you to the Tampermonkey dashboard instead.

Method B — From Tampermonkey Dashboard (always works)

1

Click the Tampermonkey icon

In your browser toolbar, click the Tampermonkey icon (the black and white circle).

2

Open Dashboard

Click Dashboard from the dropdown menu.

3

Find the script and click Edit

Find Scratch XML Injector in the list. Click the pencil (edit) icon.

4

Select All and Copy

Click inside the editor, press Ctrl+A to select all, then Ctrl+C to copy.

Method C — From this wiki

The v7.0 script file is included in the wiki's GitHub repository as scratch-xml-injector.user.js. If you're hosting this wiki on GitHub Pages, you can find the raw file at:

URL
https://YOUR-USERNAME.github.io/scratch-xml-wiki/scratch-xml-injector.user.js

Or from the GitHub repo directly:

URL
https://raw.githubusercontent.com/YOUR-USERNAME/scratch-xml-wiki/main/scratch-xml-injector.user.js

Tampermonkey can install scripts directly from raw URLs — just visit the URL and Tampermonkey will prompt you to install.

03Installing the Script from Copied Source

1

Open Tampermonkey Dashboard

Click the Tampermonkey icon → Dashboard.

2

Create a new script

Click the + tab at the top of the Dashboard.

3

Delete the default content

Select all (Ctrl+A) and delete the placeholder code.

4

Paste and save

Paste the copied script (Ctrl+V), then save with Ctrl+S. The script is now active.

✓
To update an existing installation, open the edit view for the old script, select all, paste the new source, and save. The old script is replaced in-place.
