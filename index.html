Scratch XML Injector · v6.3

Inject code into
Scratch & TurboWarp

A powerful browser userscript that lets you programmatically create sprites, costumes, and block scripts using XML — without any drag-and-drop.

Get started → Browse the docs
📝
Blocks XML
Inject standard Blockly XML directly into the active sprite's workspace — any script, any complexity.
🏗
Project XML
Bootstrap entire projects from a single XML document — sprites, auto-rendered costumes, and scripts all at once.
📤
Export XML
Read the current sprite's Blockly XML back out of Scratch to copy, edit, and reuse your blocks anywhere.
🎨
Shape Costumes
Auto-generate circle and rectangle costumes from simple XML attributes — no image files required.
🔧
Non-ASCII Fix
One-click stripping of non-ASCII characters that cause Scratch's internal asset validator to reject blocks.
⚡
TurboWarp Ready
Works on both scratch.mit.edu and turbowarp.org — same script, same XML.

Quick Start

<!-- Paste into the injector panel in "Blocks XML" mode -->
<xml xmlns="https://developers.google.com/blockly/xml">

  <block type="event_whenflagclicked" x="60" y="60">
    <next>
      <block type="control_forever">
        <statement name="SUBSTACK">
          <block type="motion_turnright">
            <value name="DEGREES">
              <shadow type="math_number">
                <field name="NUM">5</field>
              </shadow>
            </value>
          </block>
        </statement>
      </block>
    </next>
  </block>

</xml>
<!-- "Project XML" mode — creates sprites + costumes + blocks -->
<project>
  <stage color="#0a0a1a"/>

  <sprite name="Player" x="0" y="0">
    <costume type="circle" color="#4c8fff" size="40"/>
    <blocks>
      <xml xmlns="https://developers.google.com/blockly/xml">
        <!-- block scripts here -->
      </xml>
    </blocks>
  </sprite>
</project>
// Export tab → click "Grab XML" to read the current sprite's blocks
// The XML output looks like this — you can edit it and re-inject it:

<xml xmlns="https://developers.google.com/blockly/xml">
  <block type="event_whenflagclicked" x="40" y="40">
    <!-- ... your existing blocks ... -->
  </block>
</xml>

// Then click "Copy" to get it in your clipboard.
